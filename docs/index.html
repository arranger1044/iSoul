libxml++

Introduction

libxml++ is a C++ wrapper for the libxml XML parser library. It's original author is Ari Johnson and it is currently maintained by Christophe de Vienne and Murray Cumming.

Download

You can download libxml++ from here.

Required Libraries

  • For libxml++ 1.0
    • libxml2 >=2.5.8
  • For libxml++ 2.6
    • libxml2 >=2.6.1
    • glibmm-2.4 >=2.3.0 (or a subset of it containing at least Glib::ustring)
